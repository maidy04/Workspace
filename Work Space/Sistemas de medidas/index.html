Medidas Absolutas en CSS

Las medidas absolutas son unidades fijas que no cambian según el contexto. Ejemplos comunes son px, cm, y in.

Medidas Relativas en CSS

Las medidas relativas dependen de otros valores, como el tamaño de fuente del padre o el tamaño de la ventana. Ejemplos son em, rem, %, vw, y vh.

Unidades Especiales en CSS

Existen unidades especiales como ch para el ancho del carácter "0", ex para la altura de la letra "x" y fr para fracciones en grids CSS.
